# Benchmarks

There is no single top-level runner. Each subfolder has its own entrypoint and docs:

- **[`sapien/`](sapien/README.md)** — Python/SAPIEN GPU PhysX benchmarks (`python3 -m benchmark.sapien.run`).
- **[`physx_snippets/`](physx_snippets/README.md)** — PhysX snippet PTX A/B workflow and shell helpers.
